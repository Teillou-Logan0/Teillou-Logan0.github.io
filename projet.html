Gestion Séminaire

Une application de gestion pour un séminaire avec différentes conférences, utilisant des fichiers JSON pour le stockage.

Partie 1 : Gestion des inscriptions

Développement d’un formulaire permettant d’inscrire les participants en vérifiant leurs informations via un fichier JSON.

Partie 2 : Gestion des conférences

Implémentation d’un module permettant de sélectionner les conférences par créneau horaire.

Partie 3 : Exportation des données

Fonctionnalité pour exporter les données en fichier CSV pour les organisateurs.
