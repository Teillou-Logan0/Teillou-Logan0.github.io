Projet Gestion Seminaire

Description et contexte : Un séminaire est prévu le 12 octobre 2222 autour des questions urbaines, il est ouvert aux acteurs des seules municipalités, qui peuvent venir de différents départements. Ce séminaire, durant une seule journée, prévoit différentes conférences qui pour certaines ont lieu sur le même créneau horaire. Les données sont stockées dans des fichiers JSON ; ainsi, pour utiliser l’application, il ne sera pas nécessaire d’installer une base de données. Le site Web sera développé avec du code PHP et permet d'exploiter les formulaires de présentation. Des fonctions permettront d’accéder aux données dans un répertoire dédié. Leurs spécifications (signatures, services) sont présentées sous un format HTML.
